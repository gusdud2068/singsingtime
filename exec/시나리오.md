# 1. 회원가입 및 로그인
```bash
1. 우측 상단 회원 가입 버튼을 통해 회원가입 진행
   
2. 회원 가입 후 우측 상단 로그인 버튼을 통해 로그인 진행
   
3. 로그인을 해야 서비스를 이용 가능
```

# 3. 노래방
```bash
### 방 접속
1. 로그인 이후 홈화면에서 하단의 생성된 방에 접속하거나, 좌측 사이드바에서 create버튼으로 방 생성 후 접속 가능

### 방 이용

1. 최초 챔피언은 방장이 맡고, 나머지 접속자는 관중으로 접속
   
2. 접속자는 화면 중앙 도전 버튼을 통해 도전자가 되고, 도전자가 있으면 대기자 상태로 관리
   
3. 도전자 상태에서 화면 중앙 대결 버튼을 통해 곡선정을 하고 챔피언이 대결을 수락하면 노래 대결이 시작
   
4. 노래 중에 관중은 투표를 하고 챔피언과 도전자는 파트를 번갈아가며 대결을 진행 (버튼은 노래가 시작되면 대결자 아래에 생성)
   
5. 노래가 끝나면 관중의 투표 결과를 종합하여, 챔피언이 결정되고 한 대결이 끝남
   
6. 노래방에서 노래를 진행하며 다양한 기능을 이용
    * 화면 중앙 필터 버튼을 이용하여 사용자의 얼굴을 가린채 진행할 수 있음

    * 화면 중앙 녹화 기능을 이용하여 전체 노래방 화면과, 음성을 녹화할 수 있음

    * 화면 중앙 채팅 기능을 이용하여 접속자들끼리 채팅을 나눌 수 있음

    * 화면 중앙 에코 기능을 노래에 이용할 수 있음
```
